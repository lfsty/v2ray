# v2ray 一键部署

原文地址（旧版本）：https://toutyrater.github.io/  
仓库地址（新版本）：https://github.com/v2fly/fhs-install-v2ray

仅供自己学习使用，Ubuntu18.0.4+,不定期维护

一键安装
```
bash <(curl -s https://raw.githubusercontent.com/lfsty/v2ray/master/install.sh)
```

安装（包含v2ray的自带转发规则）
```
./install.sh
```
卸载
```
./uninstall.sh
```
